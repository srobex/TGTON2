# Stage 6 — Плагины и масштабирование

## Контексты плагинов
- Loader передаёт в `init_plugin(context)` объекты `dp`, `bot`, `settings`, а также core-сервисы.
- Один плагин = одна цепь. Импортируйте свои роутеры, сервисы, price feed и т.д.

## Масштабирование
- Перевод кешей на Redis (`CACHE_BACKEND=redis`).
- Вынос price feed/ Gem Hunter в отдельные воркеры (отдельные процессы + очереди).
- Настройка очередей (RabbitMQ/Kafka) для бродкастов при большом онлайне.
